# BookFlowPro Supabase (Local-first scaffolding)

This repo currently uses SQL migrations and (optional) Edge Function source files.
If you later install the Supabase CLI, you can move/adjust these files into a standard Supabase project layout.

## Folders
- `migrations/` : versioned SQL files (schema, RLS, RPC functions, reference seeds)
- `functions/` : Edge Functions source code (currently a notification dispatcher stub)
